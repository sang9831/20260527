# Kevin De Bruyne Manchester City Tribute Website

A tribute website celebrating the legendary career of Kevin De Bruyne at Manchester City FC.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Interactive timeline of De Bruyne's Manchester City career (2015-2025)
- Detailed statistics and achievements
- Gallery with lightbox functionality
- Mobile-friendly navigation menu
- Modern UI with Manchester City colors (sky blue and navy)

## How to Use

1. Replace the placeholder images in `assets/img/` with actual images
2. For production, consider optimizing images for web use
3. The site uses Font Awesome for icons - make sure to include a valid Kit ID or use the CDN version
4. Deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Customization

- Colors: Edit the CSS variables in `css/style.css` (primary: #6CABDD sky blue, secondary: #00205B navy)
- Content: Edit the text in `index.html` to update biographical information, stats, etc.
- Images: Replace placeholder images with actual photos of Kevin De Bruyne
