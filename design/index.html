Roadmap

학습 테크트리

기초 개념을 건너뛰지 않고, 취업과 코딩테스트에서 바로 활용할 수 있는 순서로 배치했습니다.

01

복잡도 분석

시간 복잡도, 공간 복잡도, 빅오 표기를 읽고 풀이 선택 기준을 세웁니다.

  • 최악/평균/최선
  • 입력 크기 감각
  • 제한 조건 해석
02

구현과 시뮬레이션

문제 조건을 코드로 옮기는 연습을 통해 실수를 줄이고 속도를 올립니다.

  • 조건 분기
  • 문자열 처리
  • 입출력 최적화
03

배열과 해시

가장 자주 쓰는 기본 자료구조를 익히고, 빠른 조회와 집계 패턴을 잡습니다.

  • 인덱스 접근
  • Map / Set
  • 빈도 계산
04

스택과 큐

괄호 검사, BFS, 최근성 유지 같은 패턴을 빠르게 구분할 수 있게 만듭니다.

  • 후입선출 / 선입선출
  • Deque 활용
  • 모니터링 패턴
05

연결 리스트와 트리

포인터 구조와 계층적 탐색을 이해해 복잡한 구조를 읽는 힘을 기릅니다.

  • 이진 트리
  • BST 개념
  • 재귀 순회
06

정렬과 이분 탐색

정렬 기준을 다루고, 이분 탐색을 응용해 조건 만족 구간을 찾습니다.

  • 비교 기반 정렬
  • 파라메트릭 서치
  • lower / upper bound
07

그래프 탐색

DFS, BFS, 최단 경로, 위상 정렬을 통해 탐색형 문제를 구조적으로 풉니다.

  • 인접 리스트
  • 방문 처리
  • 가중치 그래프
08

동적 계획법

점화식 작성, 상태 정의, 메모이제이션으로 반복 하위문제 최적화를 익힙니다.

  • Top-down / Bottom-up
  • 상태 전이
  • 테이블 설계
09

모의고사

제한 시간 안에 푸는 훈련을 반복해 문제 해석 속도와 코드 작성 습관을 만듭니다.

  • 타이머 훈련
  • 난이도 분배
  • 정답률 추적
10

문제 복기

틀린 문제를 다시 설명할 수 있도록 오답 노트와 풀이 메모를 남깁니다.

  • 실수 유형 분류
  • 대안 풀이 정리
  • 핵심 패턴 기록
11

설명 훈련

왜 그 풀이를 선택했는지 말로 설명하는 연습을 통해 면접 대응력을 높입니다.

  • 핵심 아이디어
  • 복잡도 근거
  • 엣지 케이스
12

실전 스프린트

목표 회사 문제 유형을 분석해 집중 훈련하고, 지원 일정에 맞춰 마무리합니다.

  • 기업별 경향
  • 약점 보완
  • 최종 점검
Categories

핵심 주제 분류

각 주제를 독립적으로만 보지 말고, 어떤 문제 유형과 연결되는지 같이 익히는 것이 중요합니다.

자료구조

배열, 스택, 큐, 해시, 트리, 힙, 그래프를 문제 유형별로 구분합니다.

배열 트리 그래프

알고리즘

정렬, 탐색, 분할 정복, 그리디, DP, 최단 경로를 패턴으로 익힙니다.

DP 그리디 BFS

실전 전략

문제 읽기, 시간 배분, 디버깅, 복기, 설명까지 시험장의 흐름을 대비합니다.

복기 시간관리 면접
Practice

실전 체크리스트

문제 풀이 루틴

1

조건 파악

입력 범위와 제한 시간을 보고 적합한 자료구조와 알고리즘 후보를 좁힙니다.

2

초안 구현

정답률이 높은 간단한 접근부터 코드로 옮기고, 예외 케이스를 바로 확인합니다.

3

복잡도 개선

시간 초과가 보이면 이분 탐색, 해시, DP, 그래프 탐색 등으로 개선합니다.

4

오답 복기

왜 틀렸는지, 어떤 패턴을 놓쳤는지 기록해서 다음 문제에서 반복하지 않도록 합니다.

면접용 설명 포인트

  • 이 문제에서 어떤 자료구조를 선택했는가
  • 시간 복잡도를 어떻게 계산했는가
  • 실수 가능성이 높은 엣지 케이스는 무엇인가
  • 다른 풀이와 비교했을 때 장단점은 무엇인가
  • 코드 개선 여지가 있다면 어디인가
Plan

12주 학습 플랜

한 번에 많은 문제를 푸는 대신, 각 주차마다 반드시 설명 가능한 결과물을 남기도록 구성했습니다.

1-3주

기초 고정

복잡도, 구현, 문자열, 배열, 해시를 집중적으로 다룹니다.

4-6주

구조 정복

스택, 큐, 트리, 힙을 패턴 중심으로 정리합니다.

7-9주

알고리즘 확장

정렬, 탐색, 그래프, 그리디, DP를 문제 유형과 연결합니다.

10-12주

실전 마감

모의고사, 복기, 설명 훈련으로 채용 단계에 맞춰 마무리합니다.
